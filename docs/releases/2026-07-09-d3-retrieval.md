## 探针实测(修复方案由数据选择)

- **翻译桥天花板**:中文探针改用英文孪生嵌入 → hit@3 0.5→0.867、AUROC 0.719→0.985,**完全追平英文**
- **NNN 第三次否决**:hit@3 反降(0.867→0.767-0.833)、AUROC 降 —— R1(1.7k 行)、本轮(4k 行)结论一致,此路线关闭
- **阈值原料**:英文空间 正例 top1 p10=0.729 / 负例 max=0.733 —— 分布相贴,单阈切不净 → 双阈

## 修复(pipeline,三处)

1. **跨语言桥**(node_executor):中文查询 → flash 温度0 译成动名词短语(关思考 —— 思考 token 会吃光输出上限,实测)→ 双路检索;**英译路有 strong 就以英译路为准**(两路分数刻度不同,按分高合并会把毒瘤排第一,实测教训),原文路只补中文片段(analyze 摘要)的命中;翻译失败 fail-open 单路
2. **拟合双阈**(semantic_index):`T_LO=0.70 / T_HI=0.735`(env 可调),relevance 三档 strong/borderline/weak;旧 WEAK_THRESHOLD=0.6 退役为 T_LO 别名
3. **borderline 信封**(node_executor):只有模糊带命中时不给行列表,note 指示"先核对 video_facts 再下结论"—— 结构性防"把最近邻当找到了"

## 顺带:facts 真值审计(C 批,~$1.2)

抽 42 条 matched=1 facts(按置信分层)→ flash 看片核验 + 争议样本全片复审:

- **谓词精度 ≈ 85%**(33/39;最初 65% 是审计员被时间戳锚定误导)
- **时间戳错位 ≈ 18%**(事件在 0:20,span 标在开头一类)—— 影响跳转/证据条,待单开批次治
- **置信度字段与真伪不相关**(低档 71% / 中档 54% / 高档 69%)—— 任何下游都不要拿 confidence 当质量信号
